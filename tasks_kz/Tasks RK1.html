1. Сдвиг массива / Массивті жылжыту

Русский: Дан целочисленный массив A длиной n и число x. Необходимо сдвинуть элементы массива на x шагов вправо. Если x отрицательно — сдвинуть влево. Сдвиг циклический (элементы не теряются).

Қазақша: Бүтін сандардан тұратын A массиві мен x саны берілген. Массив элементтерін x қадамға оңға жылжытыңыз. Егер x теріс болса — солға жылжытыңыз. Жылжыту циклдік түрде орындалады.

Ограничения / Шектеулер: 1 ≤ n ≤ 10⁵, −10⁹ ≤ Aᵢ ≤ 10⁹, −10⁹ ≤ x ≤ 10⁹

Формат ввода / Кіріс форматы

n x
A₁ A₂ ... Aₙ

Формат вывода / Шығыс форматы

Сдвинутый массив / Жылжыған массив

Примеры / Мысалдар

Input:
5 2
1 2 3 4 5

Output:
4 5 1 2 3
Input:
5 -1
10 20 30 40 50

Output:
20 30 40 50 10

2. Самый длинный отрезок из простых чисел / Ең ұзын жай сандар тізбегі

Русский: Дан массив целых чисел длиной n. Найдите длину самого длинного непрерывного подотрезка, состоящего только из простых чисел. Если таких нет — выведите 0.

Қазақша: Ұзындығы n болатын бүтін сандар массиві берілген. Тек жай сандардан тұратын ең ұзын үздіксіз тізбектің ұзындығын табыңыз. Егер ондай жоқ болса — 0 шығарыңыз.

Ограничения / Шектеулер: 1 ≤ n ≤ 10⁵, |Aᵢ| ≤ 10⁶

Формат ввода / Кіріс форматы

n
A₁ A₂ ... Aₙ

Формат вывода / Шығыс форматы

Длина отрезка / Тізбектің ұзындығы

Примеры / Мысалдар

Input:
10
1 2 3 5 4 7 11 13 15 17

Output:
3
Input:
6
4 6 8 9 10 12

Output:
0

3. Сокращённое представление строки / Жолды қысқарту

Русский: Дана строка s и число k. Если длина строки больше k, выведите в виде: первый символ + (длина − 2) + последний символ. Иначе выведите строку без изменений.

Қазақша: s жолы және k саны берілген. Егер жолдың ұзындығы k-дан үлкен болса, оны мына түрде шығарыңыз: бірінші әріп + (ұзындығы − 2) + соңғы әріп. Әйтпесе жолды өзгертпей шығарыңыз.

Ограничения / Шектеулер: 1 ≤ k ≤ 10⁴

Формат ввода / Кіріс форматы

s
k

Формат вывода / Шығыс форматы

Преобразованная строка / Қысқартылған жол

Примеры / Мысалдар

Input:
SERVER
4

Output:
S4R
Input:
HELLO
10

Output:
HELLO
