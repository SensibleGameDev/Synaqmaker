1. Архипелаг

В Тихом океане расположен живописный архипелаг, состоящий из N островов. Правительство выделило средства на строительство M мостов между некоторыми парами островов. Каждый мост — двусторонний.

Министр транспорта хочет узнать: можно ли после постройки всех мостов попасть с любого острова на любой другой, двигаясь только по мостам. Если это возможно, архипелаг считается единой транспортной системой.

Формат входных данных

В первой строке заданы два целых числа N и M (1 ≤ N ≤ 10⁵, 0 ≤ M ≤ 2·10⁵) — количество островов и мостов. В следующих M строках заданы пары целых чисел ui и vi (1 ≤ ui, vi ≤ N, ui ≠ vi), обозначающие мост между островами.

Формат выходных данных

Выведите YES, если из любого острова можно добраться до любого другого. Иначе выведите NO.

Пример 1
Ввод
4 3
1 2
2 3
3 4
Вывод
YES
Пример 2
Ввод
5 3
1 2
1 3
4 5
Вывод
NO
Время: 1 с  ·  Память: 256 Мб

2. Самая длинная цепочка чётных чисел

Дан список целых чисел A длины n. Требуется найти длину самой длинной непрерывной подпоследовательности, состоящей только из чётных чисел.

Формат входных данных

Первая строка содержит целое число n (1 ≤ n ≤ 2·10⁵) — длину списка. В следующей строке заданы n целых чисел — элементы A₁ A₂ … Aₙ. Для каждого |Aᵢ| ≤ 10⁹.

Формат выходных данных

Выведите длину самой длинной непрерывной последовательности чётных элементов. Если чётных чисел нет — выведите 0.

Пример
Ввод
7
1 2 4 6 3 8 10
Вывод
3
Время: 1 с  ·  Память: 256 Мб

3. Кенгуру — лестница

Кенгуру хочет добраться до верхней ступеньки лестницы. За один прыжок он может подняться на одну или две ступеньки. Всего лестница имеет n ступенек.

Вопрос: Сколько различных способов существует, чтобы кенгуру добрался до вершины лестницы?

Формат входных данных

Одно целое число n (0 ≤ n ≤ 10⁶) — номер верхней ступеньки.

Формат выходных данных

Выведите количество способов добраться до вершины лестницы.

Пример
Ввод
3
Вывод
3
