1. Архипелаг

Тынық мұхитында N аралдан тұратын көрікті архипелаг орналасқан. Үкімет кейбір аралдар жұптарының арасына M көпір салуға қаражат бөлді. Әр көпір — екіжақты (яғни екеуара жүре алады).

Көпірлер салынғаннан кейін архипелагтың кез келген аралынан кез келген басқа аралға тек көпірлер арқылы жетуге бола ма — соны анықтау қажет. Егер солай болса, архипелагты біртұтас көлік жүйесі деп атауға болады.

Енгізу форматы

Бірінші жолда екі бүтін сан N және M (1 ≤ N ≤ 10⁵, 0 ≤ M ≤ 2·10⁵) берілген — аралдар саны және көпірлер саны. Келесі M жолда ui және vi (1 ≤ ui, vi ≤ N, ui ≠ vi) бүтін сандар жұптары беріледі — сәйкес аралдар арасында көпір бар.

Шығару форматы

Егер әрбір аралдан әрбір аралға көпірлер арқылы жетуге болса — шығару YES. Әйтпесе — NO.

2. Ең ұзын жұп сандар тізбегі

Сізге ұзындығы n болатын бүтін сандардың A тізімі берілген. Тізімдегі жұп сандардан ғана тұратын ең ұзын үздіксіз тізбектің ұзындығын анықтаңыз.

Енгізу форматы

Бірінші жолда бір бүтін сан n (1 ≤ n ≤ 2·10⁵) — тізімнің ұзындығы. Келесі жолда n бүтін сан — элементтер: A₁ A₂ … Aₙ. Әрбір элементтің шамасы |Aᵢ| ≤ 10⁹.

Шығару форматы

Жауап ретінде ең ұзын үздіксіз жұп элементтерден тұратын тізбектің ұзындығын шығарыңыз. Егер жұп сан жоқ болса — 0 шығарыңыз.

Шектеулер
  • n ≤ 2·10⁵
  • Уақыт шегі: 1 секунд, жад: 256 Мб

3. Кенгуру — баспалдақ

Кенгуру баспалдақтың ең жоғары сатысына секіргісі келеді. Ол бір қадамда не бір саты (1) не екі саты (2) секіре алады. Баспалдақтың ең жоғарғы сатысы n-ге тең.

Сұрақ: Кенгуру баспалдақтың жоғарғы сатысына жетудің неше түрлі тәсілі бар?

Енгізу форматы

Бір бүтін сан n (0 ≤ n ≤ 10⁶) — баспалдақтың жоғарғы сатысы.

Шығару форматы

Баспалдақтың жоғарғы сатысына жету тәсілдерінің санын шығарыңыз.
